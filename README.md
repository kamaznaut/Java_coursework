# Java_coursework

Liberumgramm - разработка декстопного приложения для обмена текстовыми сообщениями 
на основе языка java и присущих ему технологий , главная цель минимально жизнеспособный продукт.
________________________________________________________________________________________________________________

Техническое Задание:
1.Изучить основы клиент-серверного взаимодействия в Java (сокеты, потоки, обработка данных).

2.Спроектировать архитектуру приложения, включающую серверную и клиентскую части.

3.Реализовать передачу и приём сообщений между пользователями в реальном времени.

4.Создать графический интерфейс пользователя (GUI).

5.Обеспечить хранение данных пользователей (логины, пароли, история сообщений).

6.Провести тестирование приложения на корректность и устойчивость к ошибкам.
